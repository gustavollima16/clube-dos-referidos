Conectar ao GitHub

Para publicar alterações direto no site, você precisa de um Token do GitHub com permissão de escrita no repositório.

Como gerar o token:
1. Acesse github.com → Configurações → Developer Settings
2. Personal access tokens → Tokens (classic) → Generate new token
3. Marque a permissão repo (acesso completo)
4. Gere e cole o token abaixo
